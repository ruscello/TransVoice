テキストに入力された文章を行ごとに音声ファイル(mp3形式)に変換し出力します。

## Description

text.txtに音声ファイルに変換したい文章を書くだけです！
行ごとにファイルが生成されます。

## Features

- 論文の動画作成に活用できる
- 英語の勉強に活用できる

## Usage

1. text.txtに音声ファイルに変換したい文章を書く
2. ruby say.rbと実行する
3. say.rbと同じディレクトリに生成された音声データができる
